gfalFS

Description

gfalFS is a FUSE module for GFAL 2.0. gfalFS is able to mount any point of the CERN's stockage systemnbsp;like a local folder.

Install and configure

On Scientific Linux, Fedora or Enterprise Linux:

  • Enable the EPEL repository
  • sudo yum install gfal2-all gfalFS
  • sudo usermod -a -G fuse [my login name]
  • su [my login name] (or log out and log in again)
  • gfalFS /my_mount_point [my url]

Checkout the code

Registered
svn co https://svn.cern.ch/reps/lcgutil/gfalFS/trunk
Anonymous
svn co http://svn.cern.ch/guest/lcgutil/gfalFS/trunk

Example

# mkdir ~/mount_tmp 
# gfalFS ~/mount_tmp lfc://prod-lfc-shared-central.cern.ch/grid/dteam
# ls ~/mount_tmp/
# cat ~/mount_tmp/testread0011
# gfalFS_umount ~/mount_tmp
